## Install
```
yarn add -D @npm.piece/eslint-config-npmpiece
```

```
npm i -D @npm.piece/eslint-config-npmpiece
```

## Configuration
#### create `.eslintrc` near `package.json` file and paste

```json
{
  "extends": [
    "@npm.piece/eslint-config-npmpiece"
  ]
}

```
